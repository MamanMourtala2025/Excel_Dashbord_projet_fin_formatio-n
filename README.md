Projet de Fin de Formation – Dashboard Excel pour l’Analyse de Données

📌 Présentation du projet

Ce dépôt contient le tableau de bord interactif réalisé dans le cadre de notre projet complet de fin de formation en Analyse de Données avec Excel.

L’objectif de ce projet est de simuler un cas réel d’entreprise en transformant des données brutes de ventes en indicateurs de performance clairs et en un dashboard interactif permettant d’analyser les performances commerciales et d’aider à la prise de décision.

Ce projet a été conçu comme un exercice pratique complet pour mettre en application les principales compétences d’un Data Analyst avec Excel.

---

🎯 Objectifs du projet

- Nettoyer et préparer les données
- Construire un modèle de données professionnel
- Créer des indicateurs de performance (KPI)
- Analyser les ventes, le profit et les remises
- Concevoir un tableau de bord interactif et lisible
- Utiliser Excel comme outil de Business Intelligence

---

🛠️ Outils et technologies utilisés

- Microsoft Excel
- Power Query → nettoyage et transformation des données
- Power Pivot → modélisation et relations entre tables
- DAX → création de mesures et KPI
- Tableaux Croisés Dynamiques
- Segments & Timeline
- Dashboard interactif

---

📂 Contenu du projet

Le projet comprend :

- le fichier Excel du dashboard
- les tables préparées pour la modélisation
- les indicateurs de performance
- les graphiques et visualisations
- les filtres interactifs (segments et chronologie)

---

📈 Indicateurs analysés

Le tableau de bord permet de suivre notamment :

- Total des ventes
- Profit total
- Quantité vendue
- Nombre de commandes
- Évolution des ventes dans le temps
- Performance par catégorie
- Profit par région
- Impact des remises sur la rentabilité

---

🧩 Structure du projet

Le dashboard repose sur une logique de modélisation simple et professionnelle avec plusieurs tables :

- Table Ventes
- Table Produits
- Table Clients
- Table Régions / Géographie
- Table Dates

Ces tables sont reliées dans le modèle de données afin de faciliter l’analyse multidimensionnelle.

---

🚀 Résultat attendu

À la fin du projet, on obtient un dashboard Excel interactif permettant de :

- filtrer les analyses par région, client, catégorie et période
- visualiser rapidement les performances commerciales
- analyser l’évolution des ventes et du profit
- identifier l’impact des remises sur la rentabilité
- produire un reporting clair et professionnel
